# Party Pour - Drinking Game Webapp

A responsive webapp with a full party lineup of interactive drinking games on one main page.

## Games Included

### Beer Pong

- Starts with 10 cups per team
- Tap score buttons when a team sinks a cup
- Includes per-team undo and full reset

### Ring of Fire

- Full 52-card shuffled deck
- Draw one card at a time
- Each rank maps to a classic Ring of Fire rule
- Reset reshuffles and starts a fresh deck

### Never Have I Ever

- Random rotating prompts
- One-tap prompt generation for fast rounds

### Most Likely To

- Random scenario generator
- Great for group votes and tie-break drinks

### Truth or Drink

- Random question prompts
- Players answer honestly or drink

### Flip Cup Race

- Team A and Team B flip counters
- Built-in race timer with start, stop, and reset

### Power Hour

- Manual round tracker from 0 to 60
- Includes completion status and reset

### Higher or Lower

- Guess if next card is higher or lower
- Uses a shuffled deck and live feedback per guess

## Run Locally

Open `index.html` in your browser.

If you want a local server:

```bash
python3 -m http.server 8080
```

Then open:

http://localhost:8080
